# İş Başvurum

Kullanıcıların iş başvurularını kolayca yapabilmesi ve yönetebilmesi için geliştirilmiş bir mobil uygulama.

## 🚀 Özellikler
- Kullanıcı dostu başvuru formu
- CV yükleme desteği
- Başvuru durum takibi
- Otomatik Mail gönderimi

## 🛠️ Kullanılan Teknolojiler
- Flutter
- Dart
- Firebase
